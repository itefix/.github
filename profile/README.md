## Itefix Software

Itefix Software delivers proven open source technologies packaged and supported for
Windows environments. Our products — including **cwRsync** (rsync for
Windows) and **Copssh** (SSH server and client) — have served businesses
worldwide since 2002.

This GitHub organization is our official presence. Product downloads,
licensing, documentation and support are available at
**[itefix.net](https://www.itefix.net)**.

### Products

- **[Copssh Server](https://www.itefix.net/copssh-server)** — OpenSSH/SFTP Server for Windows with GUI
- **[cwRsync Server](https://www.itefix.net/cwrsync-server)** — Rsync Server for Windows with GUI
- **[Rsync Client Helper](https://www.itefix.net/rsync-client-helper-gui)** — GUI for building and running rsync commands
- **[Secure Rsync Server](https://www.itefix.net/cwrsync-copssh-server-kit)** — cwRsync Server + Copssh Server kit for rsync over SSH on Windows
- **[Win2ban](https://www.itefix.net/win2ban)** — Fail2ban for Windows
